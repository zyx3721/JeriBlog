# 更新日志

本文档记录 JeriBlog 项目的所有重要变更。

## [2026-04-17] - 最新版本

### 用户体验优化

#### 代码块显示优化
- 移除代码块的最大高度限制（原 60 行限制）
- 移除垂直滚动条，代码块内容完整展示
- 保留横向滚动功能，处理超长代码行
- 优化滚动事件处理，仅处理横向滚动

#### 加载动画逻辑重构
- 参考 gin-vue3-blog 项目实现真正的数据驱动加载
- 新增全局加载状态管理（useAppLoading）
- 加载动画真正等待数据加载完成后才关闭
- 首次访问显示加载动画，后续路由切换不再显示
- 使用 hasInitialized 标记避免重复初始化
- 最少显示 1 秒加载动画，提升用户体验

#### 目录交互优化
- PC 端目录支持鼠标悬停滚动
- 添加 `overscroll-behavior: contain` 防止滚动穿透
- 优化滚动条样式，提升视觉体验

#### 移动端目录优化
- 减小抽屉宽度至 75%（原 80%），最大 280px（原 320px）
- 优化标题和内边距，减少空间占用
  - 标题字号从 18px 调整为 16px
  - 头部内边距从 20px 调整为 16px
  - 内容区内边距从 10px 调整为 8px
  - 目录项内边距从 10px 20px 调整为 10px 16px
- 各级标题缩进优化，更紧凑合理
- 添加小屏幕（≤360px）专属优化
- 添加 `overscroll-behavior: contain` 防止滚动穿透
- 标记 `flex-shrink: 0` 确保头部不被压缩

### 性能优化

#### 重整博客端页面加载动画逻辑
- 参考 gin-vue3-blog 项目实现
- 移除 sessionStorage 持久化标记
- 简化为：默认显示加载动画，onMounted 后隐藏
- 因为 App.vue 只挂载一次，路由切换不会重新触发加载动画
- 只在首次访问或刷新页面时显示加载动画
- 内部菜单跳转不再显示加载动画

#### 优化管理后台路由缓存
- 使用 keep-alive 缓存路由组件
- 避免内部菜单跳转时重复加载和数据请求
- 保持组件状态（滚动位置、表单数据等）
- 首次访问仍会显示加载动画，后续跳转无加载动画

### 样式优化

#### 重写代码块滚动逻辑
- 修复鼠标在代码块内部无法滚动的问题
- 正确处理垂直滚动和横向滚动
- 同时调用 `preventDefault()` 和 `stopPropagation()` 阻止事件
- 智能判断滚动方向和边界状态
- 只在代码块内部有滚动空间时阻止外部滚动
- 到达边界时允许外部滚动
- 支持 shift+wheel 横向滚动

#### 优化移动端目录背景渲染效果
- 提高背景不透明度至 98%，增强内容可读性
- 添加毛玻璃效果（backdrop-filter blur），提升视觉美感
- 增强阴影效果，提升视觉层次感
- 为暗色模式单独优化背景和阴影

### 功能增强

#### 文章管理搜索功能
- 支持按标题关键词搜索
- 支持按分类筛选
- 支持按状态筛选（已发布/草稿）
- 支持多条件组合搜索
- 提供搜索和重置功能

#### 禁止删除默认超级管理员
- 后端：在 Delete 方法中添加保护逻辑，禁止删除 ID 为 1 且角色为 super_admin 的用户
- 前端：禁用默认超级管理员的删除按钮，并添加提示信息
- 防止误删系统超级管理员账户，提升系统安全性

#### 分离博客端和管理后台登录逻辑
- 博客端改用独立的 token key (blog_access_token/blog_refresh_token)
- 避免与管理后台的 token 冲突
- 实现两个系统的登录状态完全独立

#### 移动端文章目录功能
- 添加移动端目录按钮，仅在文章页且移动端显示
- 自动提取文章标题生成目录
- 支持点击跳转到对应标题
- 实时高亮当前阅读位置
- 右侧抽屉式交互，支持滑动动画

#### 文章支持在线视频嵌入
- 支持 B 站、YouTube 和本地视频嵌入
- 添加视频样式和响应式布局
- 更新 DOMPurify 配置允许视频标签

**使用方法**：

B 站视频：
```markdown
:::video bilibili BV1xx411c7mD :::
```

YouTube 视频：
```markdown
:::video youtube dQw4w9WgXcQ :::
```

本地/在线视频：
```markdown
:::video https://example.com/video.mp4 :::
```

#### 登录用户博客端删除评论
- 后端添加 is_deleted 字段到评论响应
- 前端添加删除评论功能
- 仅评论作者可删除自己的评论
- 添加删除确认弹窗
- 删除后自动更新评论计数

#### 文章导入功能优化
- 支持 Hexo 和 Markdown 两种格式导入
- 添加图片自动下载上传功能
- 添加数据来源选择（Hexo/Markdown）
- 添加图片上传开关选项
- 重构导入逻辑，统一入口方法
- 优化错误处理和结果反馈

#### 添加 KaTeX 数学公式支持
- 安装 @traptitech/markdown-it-katex 和 katex 依赖
- 在 markdown.ts 中配置 KaTeX 插件
- 添加 KaTeX 样式定义，支持行内和块级公式
- 支持公式错误提示和暗色主题

#### 添加通用错误处理页面
- 创建统一的错误页面组件
- 支持 404、403、500 等常见错误
- 提供返回上一页和返回首页功能
- 响应式设计适配移动端

#### 文章自定义 slug 字段支持
- 前端：在文章表单中添加 slug 输入框，支持自定义 URL 标识
- 后端：Create 时优先使用用户提供的 slug，为空时自动生成
- 后端：Update 时支持修改 slug，并验证新 slug 是否已存在

#### 分类/标签计数改为实时统计
- 移除 Category 和 Tag 模型的 Count 字段
- 添加 CategoryResponse 和 TagResponse DTO，包含实时统计的 Count
- 删除 Repository 层的计数增减方法
- 修改 Service 层使用实时查询替代冗余字段
- 优势：数据一致性更好，无需维护冗余字段，避免并发更新导致的计数不准确问题

#### 异常友链通知
- 添加友链异常检测和通知功能

#### 文件引用检查基础设施
- 在所有 repository 中添加 HasFileReference 方法
- FileRepository 添加 GetByStatus 方法
- FileService 添加 FileUsageChecker 结构和引用检查逻辑
- Delete 方法添加引用检查，防止误删被引用文件

#### 代码块复制功能优化
- 优化滚动条样式，支持横向和纵向滚动
- 修复复制代码时重复行的问题
- 改用遍历直接子节点的方式，只选择 class 为 line-content 的直接子元素
- 避免 querySelectorAll 选中嵌套在高亮代码中的其他元素
- 支持 HTTP 环境下的代码复制（降级使用 document.execCommand）
- 兼容所有协议环境（HTTPS、HTTP、localhost、IP 地址访问）
- 确保长代码块复制时内容完整且不重复

#### 大文章导出超时优化
- 将导出超时时间从 5 分钟增加到 10 分钟
- 优化等待提示：5 秒后显示打包提示，30 秒后显示进度提示
- 改进用户体验，避免长时间无反馈
- 适应大文章和多图片资源的导出场景

### Bug 修复

#### 修复代码块内部滚动事件冒泡问题
- 为代码块添加 wheel 事件监听器，阻止滚动事件冒泡
- 智能判断滚动边界：未到达边界时只滚动代码块内部，到达边界时允许滚动外部页面
- 解决鼠标在代码块内滚动时外部页面也跟着滚动的问题

#### 修复博客端路由切换时加载动画闪现问题
- 优化加载状态初始化逻辑，根据 sessionStorage 标记动态设置初始值
- 如果会话中已加载过，isLoading 初始值直接为 false
- 解决点击菜单栏标签等路由切换时加载动画短暂闪现的问题

#### 修复移动端文章目录显示为空
- 修正 CSS 选择器从 .article-content 改为 .markdown-content
- 修复 extractTocFromContent 和 updateActiveId 函数
- 确保移动端目录正确提取标题

#### 修复文章表单布局不均匀问题
- 统一 Slug、分类、标签三个字段为三等分布局
- 移除 form-col-1-4 和 form-col-1-2 类名
- 统一使用 form-col 类实现均匀分布

### 响应式设计优化

- 表格添加横向滚动容器支持
- 评论元信息移动端样式优化
- 页脚导航布局调整（小屏幕 2 列）
- 导航栏断点调整为 900px
