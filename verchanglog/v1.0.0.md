## 📋 版本概述

JeriBlog v1.0.0 是一个功能完整的三端分离博客系统,包含博客前台(Nuxt 4)、管理后台(Vue 3)和后端服务(Go + Gin)。本版本实现了从项目初始化到生产环境部署的完整功能闭环。

---

## ✨ 新增特性

### 核心功能

- **博客系统基础架构**
  - 三端分离架构(Server + Admin + Blog)
  - Go 1.25 + Gin + GORM + PostgreSQL 后端服务
  - Vue 3 + Element Plus 管理后台
  - Nuxt 4 + SSR 博客前台
  - JWT 认证 + OAuth2 第三方登录
  - Swagger API 文档自动生成

- **文章管理**
  - 文章 CRUD 完整功能
  - Markdown 编辑器(CodeMirror 6)
  - 文章分类和标签管理
  - 文章搜索和筛选功能
  - 自定义 slug 支持
  - 文章导入(支持 Hexo 等格式)
  - 微信公众号导出
  - Markdown 压缩包下载

- **评论系统**
  - 多级评论支持
  - 匿名评论和登录用户评论
  - 评论管理和审核
  - 评论搜索(支持中文关键词)
  - 评论导入(支持 Artalk 等格式)
  - 评论通知

- **用户系统**
  - 用户注册/登录/注销
  - OAuth 第三方登录(GitHub、Google 等)
  - 用户权限管理(超级管理员/管理员/普通用户)
  - 用户资料编辑
  - 密码找回和重置
  - 超级管理员删除保护

- **文件管理**
  - 多存储支持(本地/MinIO/阿里云 OSS/腾讯云 COS/七牛云)
  - 文件上传和删除
  - 文件引用检查
  - 文件选择对话框
  - 图片预览

- **友链系统**
  - 友链分组管理
  - 友链申请
  - 友链状态检测
  - 异常友链通知

- **动态功能**
  - 动态发布和管理
  - 支持文本/图片/链接/音乐/视频

- **统计分析**
  - 网站访问统计
  - 文章贡献图
  - 分类和标签统计
  - 访问日志记录和查询

- **系统配置**
  - 分组配置管理
  - 配置热重载
  - 上传配置动态切换

- **通知系统**
  - 邮件通知
  - 飞书通知
  - 站内通知

- **订阅功能**
  - 邮件订阅
  - RSS 2.0 订阅源
  - Atom 订阅源
  - RSS 订阅管理

- **博客前台**

  - 响应式设计
  - 页面加载动画

  - 评论管理(搜索/筛选)
  - 文件管理(搜索/筛选)
