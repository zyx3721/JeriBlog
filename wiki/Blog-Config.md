# 博客配置
博客配置是内容最丰富的标签页，包含网站信息、全局样式、社交媒体、关于页面和自定义代码模块等。
# 网站信息
### 网站标题
网站的主标题，用于邮件通知模板、RSS 订阅、首页头部大标题等多个位置
### 网站副标题
网站的简短描述，用于首页标题后缀，同时呈现为网站 Logo
### 网站标语
网站的简短标语，在友链申请页面，用于生成站点信息配置中的 description 字段
### 网站描述
用于 SEO 优化的 meta 描述，也用于 Open Graph 和 Twitter Card
### 关键词
SEO meta keywords 标签，用于 SEO 优化用，使用多个用逗号分隔
### 建站日期
网站成立的日期，用于计算建站时长
# 全局样式
### 网站Favicon
网站图标，显示在浏览器标签页
### 背景图片
博客的背景图，通过点击标题，可切换为链接模式，能够使用302重定向随机图
### 站点截图
在友链申请页面，用于生成站点信息配置中的 screenshot 字段
### 侧边栏公告
用于展示公告内容，支持 HTML 标签和内联样式
支持的标签：<br>、<p>、<strong>、<b>、<em>、<i>、<u>、<s>、<del>、<ins>、<mark>、<small>、<span>、<sub>、<sup>、<a href="..." target="...">
支持的样式：color
### 打字机文本
首页轮播显示的文本列表，JSON 数组格式，循环显示文本
# 社交媒体
### 侧边栏社交
显示在博客侧边栏的个人社交链接，JSON 数组格式（图标使用 Remix Icon 图标库）
### 页脚社交
显示在页面底部的社交链接，JSON 数组格式（图标使用 Remix Icon 图标库）
### 页脚链接
显示在页面底部右侧的链接，JSON 数组格式
# 页面配置
### 动态数量
控制博客端动态页面的动态展示条数，默认为 30，不建议数量太多，容易导致页面卡顿
### 首页布局
博客端首页文章卡片区域的布局，支持双列瀑布流和单列卡片
### 留言信
博客端留言页面展开留言信封后展示的文本，建议 8-10 行
# 关于页面配置
### 个人描述
关于页面顶部的主要描述文本
### 描述提示
在个人描述下方的补充性文本
### 展览图片
个人资料下方的装饰性图片
### 个人资料
用于博主信息的公开介绍，支持六个关键词，JSON 数组格式
### 性格类型
用于展示 16Personalities，预设全部类型可快速选择
### 座右铭
展示两行文本组成的座右铭，JSON 数组格式
### 一言
在座右铭下方的补充性文本
### 联系方式
关于页面的联系方式列表，JSON 数组格式
### 创作平台
博主的其它创作平台列表，JSON 数组格式
### 版本信息
此网站技术栈版本信息，JSON 数组格式
### 站长联盟
加入的联盟/组织列表，JSON 数组格式
### 心路历程
关于本站的介绍和成长故事
# 自定义代码
### 字体配置
自定义博客字体，格式：字体文件URL|字体名称（推荐使用千字网）
### 自定义 Head
插入到 <head> 标签的代码，支持 meta、link、script、style 标签
### 自定义 Body
插入到 <body> 标签开头的代码
# 表情包配置
### 表情包
博客评论中使用的表情包，JSON 配置文件的 URL
可使用 https://fastly.jsdelivr.net/gh/talen8/emojis/emojis.json
如需自定义可前往 https://github.com/talen8/emojis Fork 该仓库进行自定义修改
你就可以自定义为 https://fastly.jsdelivr.net/gh/<用户名>/<仓库名>/emojis.json
