# 基本配置
基本配置包含站长个人信息、备案信息和系统地址三部分。
# 站长信息
### 站长姓名
显示在博客各处的站长名称标识，用于侧边栏作者卡片、关于页面个人介绍、页脚/文章版权信息等
### 站长邮箱
站长的联系邮箱，显示在博客各页面供用户联系，如政策协议页面、友链页面等
### 站长简介
站长的个人简介，显示在博客侧边栏作者卡片中
### 站长头像
站长的头像图片，用于侧边栏作者卡片中圆形头像展示
### 站长形象
站长的形象照片，在关于页面个人介绍右侧展示
# 备案信息
### ICP备案号
ICP 备案号，显示在页脚备案信息区域，用于链接到工信部备案查询网站
### 公安备案号
公安备案号，显示在页脚备案信息区域，用于链接到公安备案查询网站
# 系统地址
以下地址在填写时需要包含 https:// 前缀，并且无法以 / 结尾
### 管理地址
管理后台的完整访问地址，用于生成后台链接。邮件通知中，管理员相关通知（新评论、反馈投诉、友链申请）使用后台地址作为站点链接
### 博客地址
博客前端的完整访问地址，用于生成文章链接、RSS 订阅。邮件通知中，用户相关通知使用博客地址作为站点链接
### 主页地址
博客首页的完整访问地址，在页脚版权信息和文章版权声明中，用于作者主页跳转
