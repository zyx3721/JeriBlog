## 4. 文章导出
点击导出按钮打开导出对话框，支持两种导出格式：
## 4.1 复制微信公众号格式
将文章复制为兼容微信公众号的 HTML 格式，会自动复制到剪贴板。
## 4.2 下载为 Markdown
导出为 ZIP 压缩包，包含：
文章原文（.md 文件）
配图文件夹（文章中使用的所有图片）
封面图
下载文件名格式为「文章标题.zip」。
注意：文件较大时系统会显示等待提示，请耐心等待下载完成。
## 5. 分类管理
点击「分类管理」按钮打开分类管理弹窗。

## 5.2 新增分类
点击列表右上角「新增」按钮，弹出分类表单：
## 5.3 编辑分类
点击分类列表中的「编辑」按钮，弹出编辑表单，预填当前分类信息。
## 5.4 删除分类
点击分类列表中的「删除」按钮，需二次确认后删除。

## 6.3 编辑标签

## 6.4 删除标签

| 功能 | 说明 |
| --- | --- |
| 新增文章 | 创建新文章，进入文章编辑页面 |
| 刷新 | 重新加载文章列表 |
| 分类管理 | 打开分类管理弹窗 |
| 标签管理 | 打开标签管理弹窗 |
| 分页 | 支持分页浏览，每页 20 条 |


| 字段 | 说明 |
| --- | --- |
| 封面 | 文章封面缩略图，尺寸 80x50 |
| 标题 | 文章标题，支持状态标签显示（置顶、精选、草稿） |
| 分类 | 文章所属分类 |
| 标签 | 文章标签列表 |
| 发布地点 | 文章发布的地理位置 |
| 统计 | 浏览量、评论数 |
| 发布时间 | 显示发布时间和更新时间 |


| 标签 | 样式 | 说明 |
| --- | --- | --- |
| 置顶 | 蓝色 | 文章置顶显示 |
| 精选 | 绿色 | 精选文章标记 |
| 草稿 | 橙色 | 未发布的文章 |
| 未发布 | 灰色 | 尚未发布 |


| 操作 | 说明 |
| --- | --- |
| 编辑 | 进入文章编辑页面 |
| 导出 | 打开导出对话框 |
| 删除 | 删除文章（需二次确认） |


| 字段 | 说明 |
| --- | --- |
| 分类名称 | 分类的显示名称 |
| 描述 | 分类的简短描述 |
| 文章数 | 该分类下的文章数量 |
| 排序 | 分类的排序权重 |
| 操作 | 编辑、删除 |


| 字段 | 说明 |
| --- | --- |
| 名称 | 分类名称（必填） |
| 描述 | 分类描述（选填） |
| 排序 | 排序权重，数字越小越靠前 |


| 字段 | 说明 |
| --- | --- |
| 标签名称 | 标签的显示名称 |
| 描述 | 标签的简短描述 |
| 文章数 | 使用该标签的文章数量 |
| 操作 | 编辑、删除 |


| 字段 | 说明 |
| --- | --- |
| 名称 | 标签名称（必填） |
| 描述 | 标签描述（选填） |
