管理后台左侧菜单进入「用户管理」，即可查看所有用户列表、进行新增、编辑、删除等操作。

## 4.3 表单预填
编辑表单会自动填充用户的当前信息：
邮箱、昵称、网站地址、铭牌标识、角色

## 4.4 更新提交
点击「确定」按钮更新用户：
表单验证通过后
如果有新头像，先上传头像
构建更新数据（密码留空则不包含）
调用更新接口
显示成功提示
关闭对话框并刷新列表
## 5. 删除用户
## 5.1 删除操作
在用户列表中点击某用户的「删除」按钮。
## 5.2 确认提示
删除前系统会弹出确认框：确定要删除这个用户吗？
## 5.3 删除类型
### 用户删除采用软删除机制：
用户数据不会从数据库物理删除
仅设置 deleted_at 时间戳标记
被删除用户在列表中显示「已删除」标签
「已删除」用户不显示编辑和删除按钮

## 6.2 权限范围

| 功能 | 说明 |
| --- | --- |
| 分页展示 | 每页显示 20 条数据 |
| 新增用户 | 创建新用户账户 |
| 刷新 | 重新加载用户列表 |
| 操作 | 编辑、删除用户 |


| 字段 | 说明 |
| --- | --- |
| 头像 | 用户头像图片，无头像则显示默认图标 |
| 昵称 | 用户昵称，支持铭牌标识显示 |
| 邮箱 | 用户注册邮箱，禁用状态会显示标签 |
| 网站地址 | 用户个人网站 URL |
| 角色 | 用户的权限角色 |
| 登录方式 | 用户已绑定的登录方式图标 |
| 最后登录 | 用户最后一次登录的时间 |


| 角色 | 标签颜色 | 说明 |
| --- | --- | --- |
| 超级管理员 | 红色 | 拥有系统全部权限 |
| 管理员 | 橙色 | 拥有管理后台大部分权限 |
| 普通用户 | 绿色 | 基础用户权限 |
| 访客 | 灰色 | 仅可浏览内容 |


| 状态 | 标识位置 | 说明 |
| --- | --- | --- |
| 已删除 | 昵称旁红色标签 | 用户已软删除 |
| 已禁用 | 邮箱旁红色标签 | 用户账户被禁用 |


| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| 邮箱 | 输入框 | 是 | 用户登录邮箱，格式验证 |
| 昵称 | 输入框 | 是 | 用户显示名称，2-32 字符 |
| 网站地址 | 输入框 | 否 | 用户个人网站 URL |
| 铭牌标识 | 输入框 | 否 | 显示在昵称旁的特殊标识 |
| 密码 | 密码框 | 是 | 账户密码，6-32 字符 |
| 头像 | 图片上传 | 否 | 用户头像图片 |
| 角色 | 下拉选择 | 是 | 用户权限角色 |


| 字段 | 验证规则 |
| --- | --- |
| 邮箱 | 必填，合法邮箱格式 |
| 昵称 | 必填，2-32 字符 |
| 网站地址 | URL 格式验证（选填） |
| 密码 | 必填，6-32 字符 |
| 角色 | 必填，从下拉框选择 |


| 差异项 | 新增 | 编辑 |
| --- | --- | --- |
| 密码字段 | 必填 | 选填（留空不修改） |
| 启用状态 | 不显示 | 显示开关 |


| 角色 | 角色标识 | 说明 |
| --- | --- | --- |
| 超级管理员 | super_admin | 系统最高权限，可管理所有用户和设置 |
| 管理员 | admin | 拥有管理后台大部分管理权限 |
| 普通用户 | user | 基础权限，可评论、发表文章等 |
| 访客 | guest | 仅可浏览公开内容 |


| 功能 | 超级管理员 | 管理员 | 普通用户/访客 |
| --- | --- | --- | --- |
| 用户管理 | 完全控制 | - | - |
| 文章管理 | 完全控制 | 完全控制 | - |
| 评论管理 | 完全控制 | 完全控制 | - |
| 系统设置 | 完全控制 | 部分 | - |
| 内容浏览 | 是 | 是 | 是 |
