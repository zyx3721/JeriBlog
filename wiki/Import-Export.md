# 导入文章
目前支持 Hexo 格式和 Markdown 格式的文章，可以开启图片处理选项，自动下载并上传文章中的图片。
最理想的格式为 Hexo 那样带 Front-matter 的 Markdown 文章，可以自动解析出其中的所有字段进行匹配，比如链接、标题、发布时间、摘要等。
## 导入评论
目前仅支持 Artalk 评论系统的评论数据，如果你的评论数据中携带访问数据，也会把访问数据一并迁移进来。
如果你的文章没有使用 Hexo 和 hexo-abbrlink 插件，那么不建议直接导入评论，会出现评论与文章无法绑定的问题。
## 导出文章
目前仅支持单篇文章的导出，位置在文章管理页面的操作列，支持将封面和文章配图下载，并把文章中的图片链接更换为本地引用，最终文章与图片打包为 ZIP 包。
如果你不需要下载图片，那么建议在文章编辑页面里复制全文。
